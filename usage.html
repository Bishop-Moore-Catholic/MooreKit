BMC V3 Usage Guide

A scoped, Bootstrap-style reference for layout, components, typography, and utility classes.

Quick Start

Load the stylesheet, then wrap framework content once with bmc-v3. Add the website's existing button class to links to avoid global link-color overrides.

<link rel="stylesheet" href="bmc-v3.css"> <div class="bmc-v3"> <section class="bmc-section"> <div class="bmc-container">...</div> </section> </div>

Layout

Containers

bmc-containerCentered container, maximum 1200px.
bmc-container-fluidFull-width container with responsive side padding.
bmc-container-narrowCentered reading-width container, maximum 850px.

Sections

bmc-section, bmc-section-sm, bmc-section-dark, bmc-section-light, and bmc-section-gold.

12-Column Grid

Use bmc-row with bmc-col-1 through bmc-col-12. Responsive forms are available at sm, md, and lg, such as bmc-col-md-6.

<div class="bmc-row bmc-g-3"> <div class="bmc-col-12 bmc-col-md-6">...</div> <div class="bmc-col-12 bmc-col-md-6">...</div> </div>

CSS Grid

bmc-grid plus bmc-grid-2, bmc-grid-3, bmc-grid-4, or bmc-grid-auto.

Alternating Grid

bmc-grid-alternate automatically places odd direct children in the left column and even direct children in the right column at 992px and wider. It collapses to one column below that breakpoint. Add bmc-grid-alternate-align to alternate right/left text alignment on large screens.

<div class="bmc-grid-alternate bmc-grid-alternate-align bmc-gap-4"> <article>Left on large screens</article> <article>Right on large screens</article> <article>Left on large screens</article> </div>

Typography

Semantic h1 through h6 receive responsive defaults inside the namespace.

ClassUse
bmc-display-1, bmc-display-2Largest campaign and hero titles.
bmc-heading-1 through bmc-heading-3Reusable visual heading sizes independent of semantic level.
bmc-heading-light, -regular, -medium, -semibold, -boldHeading-weight utilities.
bmc-heading-splitLight title with bold text inside strong or bmc-heading-emphasis.
bmc-eyebrowSmall uppercase gold section label.
bmc-lead, bmc-small, bmc-xsmallBody-size variants.

Components

Buttons

bmc-btn, bmc-btn-dark, bmc-btn-outline, bmc-btn-light, bmc-btn-sm, bmc-btn-lg, and bmc-btn-block.

<a class="bmc-btn bmc-btn-dark" href="#">Dark Button</a>

Pills

bmc-pill with bmc-pill-gold, bmc-pill-dark, bmc-pill-outline, bmc-pill-sm, or bmc-pill-lg.

Cards

bmc-card, bmc-card-dark, bmc-card-gold, and bmc-card-white.

Legacy and Image Cards

Use bmc-legacy-card for the image-and-gold-caption design. Images default to grayscale; add bmc-image-current for full color. Ratio modifiers include bmc-legacy-card-landscape and bmc-legacy-card-square.

<figure class="bmc-legacy-card"> <img src="..." alt="..."> <figcaption>Caption</figcaption> </figure>

Hexagons

Use bmc-hex for the default flat-top regular hexagon. Add bmc-hex-pointed to rotate the geometry into a point-up/point-down regular hexagon. Size and color modifiers work with either orientation.

<div class="bmc-hex bmc-hex-pointed bmc-hex-dark">...</div>

Team Timeline

Use bmc-team-timeline with alternating bmc-team-timeline-item entries. Each entry contains a media wrapper and content wrapper. Portraits can use bmc-image-circle, the flat-top bmc-image-hex, or the point-up/point-down bmc-image-hex-pointed. Add bmc-team-timeline-compact to the parent for smaller desktop portraits.

Add bmc-team-timeline-left to move the rail to the left and place all entries to its right. Wrap an existing MooreKit pill in bmc-team-timeline-year for a year or milestone label, and use bmc-team-timeline-gallery when one entry contains multiple images. The left variation collapses to a content-first single column on mobile.

<div class="bmc-team-timeline"> <article class="bmc-team-timeline-item"> <div class="bmc-team-timeline-media bmc-image-shape bmc-image-circle"> <img src="..." alt="Name"> </div> <div class="bmc-team-timeline-content"> <h3 class="bmc-team-timeline-name">Name</h3> <p class="bmc-team-timeline-role">Role</p> <p>Biography or description.</p> </div> </article> </div> <div class="bmc-team-timeline bmc-team-timeline-left"> <article class="bmc-team-timeline-item"> <div class="bmc-team-timeline-media"><img src="..." alt="..."></div> <div class="bmc-team-timeline-content"> <p class="bmc-team-timeline-year"> <span class="bmc-pill bmc-pill-gold bmc-pill-lg">1954</span> </p> <p>Milestone description.</p> </div> </article> </div>

Utilities

Text Alignment

Use bmc-text-left, bmc-text-center, bmc-text-right, bmc-text-start, or bmc-text-end. Responsive left, center, and right variants are available with sm, md, and lg breakpoints.

Display and Flex

bmc-d-*, bmc-flex-*, bmc-justify-*, and bmc-align-*.

Spacing

Margin and padding utilities use values 0 through 5: bmc-m-*, bmc-mt-*, bmc-mb-*, bmc-my-*, bmc-p-*, and related forms.

Sizing and Position

bmc-w-25, bmc-w-50, bmc-w-75, bmc-w-100, bmc-h-100, position, overflow, and visually-hidden helpers.

Global Image Shapes

Use bmc-image-shape with one shape class. These utilities are not tied to the team timeline.

<div class="bmc-image-shape bmc-image-hex">
  <img src="photo.jpg" alt="Description">
</div>

<div class="bmc-image-shape bmc-image-hex-pointed bmc-image-shape-lg">
  <img src="photo.jpg" alt="Description">
</div>

Available shapes: bmc-image-circle, bmc-image-hex, and bmc-image-hex-pointed. Available sizes: bmc-image-shape-sm, the default size, bmc-image-shape-lg, and bmc-image-shape-fluid.

Hexagon geometry note

Global hexagon image classes preserve a regular six-sided shape inside cards, grids, and the team timeline. Use bmc-image-hex for flat-top and bmc-image-hex-pointed for point-up orientation.
